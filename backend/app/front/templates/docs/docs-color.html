Color

1. Override Colors

For any change in colors : scss/variable.scss 
                  
// colors
$white: #fff !default;
$gray-100: #f6f7f9 !default;
$gray-200: #eff4fa !default;
$gray-300: #dfe5ef !default;
$gray-400: #7c8fac !default;
$gray-500: #5a6a85 !default;
$gray-600: #29343d !default;
$black: #000 !default;

// Color
